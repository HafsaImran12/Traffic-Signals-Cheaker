# 🚦 Traffic Signal Checker

A simple and interactive Traffic Signal Checker built with HTML, CSS, and JavaScript.

## ✨ Features

- 🔴 Red Signal → Displays "Stop!"
- 🟡 Yellow Signal → Displays "Start!"
- 🟢 Green Signal → Displays "Go!"
- 🎨 Dynamic traffic light color changes
- 📱 Responsive and clean UI
- ⚡ Built with Vanilla JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5

## 📸 Preview

Traffic signal lights change dynamically when different buttons are clicked.

## 🎯 Learning Objectives

This project helped practice:

- DOM Manipulation
- Event Listeners
- CSS Styling
- Class Manipulation
- JavaScript Logic
